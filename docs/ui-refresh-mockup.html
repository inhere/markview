UI Refresh Design

MarkView is a local Markdown reader for developers. The refreshed interface keeps the current workflow intact while making reading, navigation, and controls quieter.

The main change is visual hierarchy: content leads, navigation supports, controls stay available without competing for attention.

Layout

The shell keeps three stable regions: files, document, and table of contents. The center document uses a softer page surface with responsive padding so long documents stay readable.

.paper {
  border-radius: 8px;
  padding: clamp(34px, 5vw, 72px);
}

Controls

The toolbar becomes a compact control strip with segmented layout buttons and consistent icon sizing. Focus rings and touch targets are explicit.

  • Use SVG icons instead of emoji UI affordances.
  • Keep native selects for color and theme choices.
  • Disable heavy motion when reduced motion is requested.
