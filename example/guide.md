## 第一章：概览

MarkView 会把当前目录中的 Markdown 文件渲染成 HTML 页面，并通过 SSE 监听内容变化。

### 1.1 服务端职责

服务端负责：

- 读取 Markdown 文件
- 使用 Goldmark 转换为 HTML
- 注入模板
- 提供 `/sse` 实时刷新通道

### 1.2 前端职责

前端负责：

- 代码高亮
- Mermaid 渲染
- 目录生成
- 工具栏交互
