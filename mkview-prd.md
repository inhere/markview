# mkview 产品需求文档

一个轻量级、零配置的 Markdown 预览工具，能够实时渲染本地文档，支持代码高亮和流程图绘制，以便于撰写PRD、技术文档和 README。

## 核心价值

- 零配置: 开箱即用，无需安装复杂的依赖链（如 Jekyll, Hugo）。
- 实时性: 文件修改自动刷新，提供流畅的“所见即所得”体验。
- 专业性: 内置“瑞士风格”文档布局，支持 Mermaid 图表和代码高亮。

## 核心渲染引擎

 - Markdown 解析: 使用 marked 库将 Markdown 转换为 HTML。
 - 静态文件服务: 支持服务根目录下的图片、CSS 等静态资源。
 - 安全限制: 禁止访问根目录以外的文件路径（防止目录遍历攻击）。

## 页面功能与交互

 - 文档布局:
  - 采用左侧固定侧边栏（Sidebar）+ 右侧流式内容区（Content）的布局。
  - 响应式设计：在移动端（<1024px）自动隐藏侧边栏，优化阅读体验。
 - 自动目录 (TOC):
  - 根据文档中的 H1-H3 标题自动生成侧边栏目录。
  - Scroll Spy: 滚动页面时，侧边栏自动高亮当前阅读章节。
  - 点击目录项平滑滚动至对应章节。
 - 代码高亮:
  - 集成 highlight.js，支持多种编程语言的语法高亮。
  - 样式主题采用 GitHub 风格。
 - Mermaid 图表支持:
  - 自动识别 `mermaid` 代码块并渲染为图表。
  - 增强查看:
   - 每个图表右上角提供“全屏查看”按钮。
   - 点击按钮弹出模态框（Modal）全屏展示图表，支持 ESC 键关闭

## 实时预览 (Live Reload)

 - 文件监听: 监听目标目录下的 .md 文件变更。
 - 自动刷新:
  - 后端通过 Server-Sent Events (SSE) /sse 接口向客户端推送变更通知。
  - 前端接收到 reload 信号后自动刷新页面。
 - 状态指示: 侧边栏顶部显示连接状态（Connected/Disconnected/Syncing）
